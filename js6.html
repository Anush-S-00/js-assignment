Numbers

This is variable type which stores numbers in it

Example :  var x = 20

Dynamic Types

This means that the same variable can be used to hold different data types

Example :  var x = 10

Strings

A string (or a text string) is a series of characters like "Anush Dev".

Example :  var x = "Anush Dev"

Booleans

Booleans can only have two values: true or false.

Example :  var x = 10 (x==10)

Arrays

JavaScript arrays are written with square brackets. Array items are separated by commas.

Example :  var x = [10,20,30]
