Various console function in JS

Console Clear()

The console.clear() will clear the console and displays the message as  console was cleared

syntax is   console.clear();

Console Error

The console.error() method writes an error message to the console

syntax is  console.error()

Console Group

The console.group() method indicates the start of a message group.

Syntax is console.group()

Console Time

The console time function method starts a timer in the console view

Syntax is console.time()
